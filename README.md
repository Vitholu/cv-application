Resume builder features:
  
  ## Main features
  - Section to add general information e.g: name, email, phone number.
  - Section to add educational experience e.g: school name, title of study, date of study)
  - Section to add practical experience e.g: company name, position title, main tasks of your jobs, date from and until when you worked for that company.

  ## Extra features
  - Edit and submit button for each section(or the whole CV).
